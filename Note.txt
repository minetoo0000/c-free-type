# C 언어의 기본 타입은 원시 타입이라고 불린다.
# C 언어의 기본 타입에 대한 값(데이터)은 원시 데이터라고 불린다.

[정의 : Cobject에서의 타입] - Cobject에서 정의하는 타입은 1개 이상의 원시 데이터와 1개 이상의 원시 데이터에 관한 추상적인 데이터의 타입이 포함된 객체 타입을 나타낸다.
 - Cobject의 모든 타입은 let 타입으로 불려야 한다. 즉, let 객체 타입으로 통일되어야 한다.
 - 타입 변수 생성자 함수의 반환 값은 모두 let 타입이여야 한다.
 - 타입 변수 생성자 함수로 전달되는 인자값의 타입은 모두 원시 타입이여야 한다.
 - 타입의 부가 함수는 타입에 상속되어야 한다.
 	- 예 : Array$.push(array, item)

[ 2023.01.31 ]
"Array, String 타입 변수를 생성할 때 메모리를 동적 할당해 값을 저장한다. 이것이 꽤 골칫거리가 될 것 같기도 하고 새로운 타입인 Mem 타입이
추가되었으니 다른 방식으로 재정립하려한다."

"우선 Array, String 타입의 값은 해당 메모리 주소값만 그대로 복사하도록 한다. 그렇게 하면 지역 변수 값이여도, 전역 변수 값이여도
메모리 공간을 잃을 일이 없다."

"Mem 타입은 다른 타입들과는 달리 동적 할당 메모리 주소 정보가 타입에 포함된다.
즉, 선언된 let 타입 변수가 지역 변수더라도 변수의 내부에 담긴 할당되어있는 메모리 공간은 지역 변수가 아니다.
그러니까 Mem 타입은 특수한 타입이라는 것인데 그런 점이 전혀 티가 나지 않아 자칫하면 메모리 누수가 발생하기 쉬울 수 있다.
그래서 동적 할당과 상황이 비슷하다고 생각되는 토큰의 개념을 적용시켜보려고 한다. 토큰도 값을 잃어버리면 안되니 적절하다는 생각이 든다.
예시를 보자면 let MemToken( uint64_t mem_byte_size )정도 되겠다."
